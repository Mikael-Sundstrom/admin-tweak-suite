# Feature Request

Thank you for your interest in improving Admin Tweak Suite! Before submitting a request, please check if a similar suggestion already exists in our [issues](https://github.com/Mikael-Sundstrom/admin-tweak-suite/issues).

## How to Submit a Feature Request
1. **Describe your feature request clearly** – What would you like to see?
2. **Explain why this feature is useful** – How will it benefit users?
3. **Provide examples if possible** – Screenshots, mockups, or use cases help us understand your idea.

Submit your feature request [here](https://github.com/Mikael-Sundstrom/admin-tweak-suite/issues/new?template=feature_request.md).

Thank you for helping us improve the plugin! 🚀
