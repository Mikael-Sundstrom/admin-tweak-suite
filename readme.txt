== Description ==
A suite of tools to customize and optimize the WordPress admin panel. Features:
* Admin menu customization
* Custom CSS and JavaScript management
* Advanced script handling for improved performance

== Installation ==
1. Upload `admin-tweak-suite` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
