A lightweight plugin to customize and optimize the WordPress admin panel.

== Description ==
Admin Tweak Suite improves the WordPress dashboard by giving you full control over menus, CSS, scripts, and performance tweaks — without bloated frameworks or complexity.

Features:
* Admin menu customization (drag and drop)
* Custom CSS and JavaScript injection (frontend & admin)
* Script disable/cleanup options for faster load time and better privacy

== Screenshots ==
1. Customize menu order.
2. Customize frontend and admin CSS easily.
3. Manage frontend scripts and settings.

== Installation ==
1. Upload `admin-tweak-suite` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==

= Does this affect all users? =
Yes. Any CSS, menu change or script modification applies globally.

= Is it safe to disable scripts like emojis or embeds? =
Yes — these are optional features and many sites disable them for performance or GDPR reasons.
